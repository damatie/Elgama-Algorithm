
#### HOW TO USE ALGORITHM

1 INSTALL pycryptodome  PACKAGE ( https://pypi.org/project/pycryptodome/)
2 RUN THE generateKeys.py TO GENERATE KEYS ( PRIVATE, PUBLIC, AND DSA KEYS)
3 RUN THE encrypt.py TO HASH, SIGN AND ENCRYPT FILE 
4 RUN THE decrypt.py TO VERIFY HASH, SIGNATURE,AND DECRYPT FILE
5 CREATE user_1 AND user_2 FOLDER
5 REPLACE THE FILES PATH WITH YOURS
